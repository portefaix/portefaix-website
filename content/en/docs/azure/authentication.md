This section shows the how to setup Portefaix with authentication and authorization support in Microsoft Azure (AZURE)

## Configure Azure

```shell
❯ . ./portefaix.sh azure
[ Portefaix ]
Setup credentials
Done
```

## Configure kubectl

```shell
❯ make -f hack/build/azure.mk azure-kube-credentials ENV=dev
```

```shell
❯ kubectl get nodes
```
